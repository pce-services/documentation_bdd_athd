Description

Description

Cette vue permet d'ajouter et de supprimer des points techniques en étude. Il s'agit d'une sélection parmi les 3 couches présentées dans le contexte ci-dessous.

Contexte

  • pt_piquetage : Ensemble des points techniques à traiter en piquetage
  • pt_aiguillage : Ensemble des points techniques à traiter en aiguillage
  • pt_projection_gc : Ensemble des points techniques à créer, donc inexistants sur le terrain avant travaux PCE

Pour des raisons de cohérence, seuls des points techniques fiabilisés (ou demandés en fiabilisation) en piquetage / aiguillage sont utilisables dans l'étude. Si un point technique inexistant est nécessaire, il doit être explicitement indiqué comme tel, par sa présence dans la couche pt_projection_gc.

Construction de la vue

  • La couche t_pt_etude stocke la liste des pt_id à afficher ainsi que les champs étude rattachés
  • Pour chaque pt_id de cette liste : récupérer la géométrie, le nom, le gestionnaire, le code INSEE et la structure dans t_pt
