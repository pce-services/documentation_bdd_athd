Description

Description

Cette vue permet de générer automatiquement une commande d'accès en fonction des études. Une fois cette commande d'accès générée, elle la comparée à la production : commandes d'accès enregistrées dans la table ca.

Contexte

  • cm : Chemin / tronçon entre les points techniques. Chaque cm comporte un identifiant unique appelé cm_id
  • cb : Cable optique. Chaque cb est défini par une liste de cm_id. La géométrie du cb est donc généré par le groupement des géométries de chaque cm le composant
  • Une commande d'accès comporte les cm Orange utilisés en étude, multiplié par le nombre de câble. Si 3 câbles passent par un seul cm, ce dernier doit apparaitre 3 fois dans la C3A.

Construction de la vue

Pour chaque cm_id constituant un câble présent dans cb_etude :

  • Récupérer le pt_etude amont et pt_etude aval
  • Vérifier si un des deux pt_etude à un gestionnaire Orange. Si oui : conserver le cm. Si non : passer au suivant.
  • Définir les type de pt_etude (C, CT, P, PT ...) ainsi que le diamètre, la longueur, et le commentaire (nom du cable)
  • Vérifier si l'association pt_id_a / pt_id_b / commentaire existe dans la table ca

Conditions d'édition

  • Lecture : Oui
  • Modification attributaire : Non
  • Modification geométrique : Non
  • Création d'entité : Non
  • Ajout d'entité : Non
  • Suppression : Non

La modification de cette vue se réalise indirectement :

  • Ajouter ou supprimer des éléments dans l'étude peut ajouter ou supprimer des tronçons suivant le besoin en C3A
  • Modifier des éléments d'étude peut changer le commentaire, la longueur, le nom des points techniques, etc...
  • Modifier la table ca peut cocher ou décocher la case correspondance_ca

Résolution d'erreur

Voici les points à controler en cas d'erreur ou correspondance ca invalide :

  • Correspondance pt_id_a et pt_id_b identique dans ca et cm_ca_auto. Inversion de sens interdite.
  • Le commentaire de la ca doit être égal à cb_code de la table cb_etude (par conséquent, égal à la colonne commentaire de cm_ca_auto)
  • Un erreur de géométrie dans l'étude : Un cm non accroché à un pt, une géométrie de câble invalide, etc
  • Rafraichir la vue
